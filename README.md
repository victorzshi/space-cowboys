C++ clone of Taito's arcade game Space Invaders (1978).

> TODO: Add gameplay gif.

## Installation

Clone repo:

```
git clone https://github.com/victorzshi/space-invaders.git
```

Initialize submodules:

```
git submodule init
git submodule update
```

Build with CMake:

```
cmake -B build
cmake --build build
```
